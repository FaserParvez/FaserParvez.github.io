This is a project that was developed by using Java and Javaspace. The purpose of this project was to create a chatroom where users are able to communicate with each other. The data where this information would be stored was used by a system called Javaspace. Apache River was used with Linux to create this by starting up a Javaspace. The Javaspace would then be used to collect the data back to the chatroom application.

The features that this system has is an authentication login system where the user is able to either create an account or just log in if they already have one. The user is then able to view all chatrooms available on the java application and create their own chatroom. Inside a chatroom, the user is able to communicate with other users using the application and send public message. There is also an option to send a private message to the chatroom owner. Finally the user can delete their own chatroom.
