This project was created for an assignement in Web Design and Programming for my degree. The purpose of this project was to use a testing IDE to develop my testing skills rather than focusing on the the web design part. I was purposed to make a simple registration form which would have simple validation. Then the main objective was to create some automatic test cases which would check to see if the validation works.

The main part of the project was to use a testing system called Selenium IDE. With javascript and Jquery, I was able to produce some test cases and sample form inputs which allowed the Selenium to input some random data in and check if the validation is correct. The end result of this assignment helped me develop my testing skill while scoring a mark of 80% (A).
